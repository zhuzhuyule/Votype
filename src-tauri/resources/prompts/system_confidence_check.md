# 润色质量评估

评估语音转录润色结果的质量。

## 评估标准

1. **语义保留**：润色后是否保留了原始语义？
2. **准确性**：修正是否正确？（拼写、语法、专业术语）
3. **适当性**：修改幅度是否适当？（没有过度修改或添加不存在的内容）

## 输入

原始转录：
${source_text}

润色结果：
${target_text}

## 输出格式

严格返回 JSON：

{"confidence": 0-100, "reason": "简短说明，置信度低于70时必填，否则为 null"}

置信度参考：

- 90-100: 非常好，完全保留原意且明显改善
- 70-89: 较好，基本正确
- 50-69: 有疑问，可能存在语义偏差
- 0-49: 有明显问题，需要人工审查
