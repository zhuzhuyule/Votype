## 🎨 用户体验演进

### 现状 (v1.0)

```
用户操作流程:
1. 按住快捷键
2. 说话
3. 松开
4. 等待结果
5. 文本插入

问题:
• 所有场景使用同一套提示词
• 无法快速切换功能
• 不知道 AI 在做什么
```

### 目标 (v2.0)

```
用户操作流程:

方式 1: 默认使用（自动适配）
1. 按住快捷键
2. 说话
3. 松开
4. AI 根据应用自动选择 Skill
5. 看到 Skill 指示器: "✍️ 智能续写"
6. 文本插入

方式 2: 显式指定
1. 按住快捷键
2. 说 "翻译：Hello World"
3. 松开
4. AI 使用翻译 Skill
5. 看到 Skill 指示器: "🌐 智能翻译"
6. 文本插入

方式 3: 快捷键切换
1. 打开托盘菜单
2. 选择 "💻 代码生成"
3. 按住快捷键说话
4. 使用代码生成 Skill

优势:
✅ 场景自动适配
✅ 灵活切换功能
✅ 实时反馈
```
